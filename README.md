# Приложение «Цитаты» — генератор и менеджер цитат

---

## 📝 Описание

Приложение `Цитаты` — это консольная утилита на Python, которая позволяет:

- 🎲 Генерировать случайные цитаты из библиотеки
- 📚 Добавлять новые цитаты (с проверкой на пустые поля)
- 🕰 Просматривать историю сгенерированных цитат
- 🔍 Фильтровать историю по автору и по теме
- 💾 Автоматически сохранять и загружать данные в формате JSON

Программа идеально подходит для ежедневного вдохновения, изучения Python или как пример простого CRUD-приложения с файловым хранилищем.

---

## 🛠 Требования

- Python 3.6 или выше
- Стандартные библиотеки (`json`, `os`, `random`, `typing`)
